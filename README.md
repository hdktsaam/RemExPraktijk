# Rem. examen praktijk

## lees onderstaande opgaven zorgvuldig en probeer zo goed als mogelijk alles op te lossen. 

1. in eerste instantie haal je alle users op die een cart hebben (een shoppingcart)
2. na het selecteren van een user krijg je al zijn carts tezien
3. na het selecteren van een cart krijg je de producten uit die cart tezien.

Werk stap voor stap !! 

TIPS:

https://fakestoreapi.com/carts --> hiermaal haal je alle carts op -- filter daar alle users uit om je eerste selectbox te vullen

Bekijk goed wat je terugkrijgt van deze API.

Met deze ene API Call zou je er al kunnen geraken. Indien nodig kan je nog onderstaande API ook gebruiken. 

https://fakestoreapi.com/carts/1 --> dit haalt nogmaals de cart op van 1 user. 

// het volstaat om enkel de id van het product weer te geven --> dan behaal je reeds 90%
